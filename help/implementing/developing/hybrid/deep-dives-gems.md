# Immersion dans les SPA {#spa-deep-dives}

[Gems sur Adobe Experience Manager](https://helpx.adobe.com/fr/experience-manager/kt/eseminars/gems/aem-index.html) est une série de séances approfondies réalisées par des experts Adobe. Hébergée par les ingénieurs d’Adobe, cette série peut vous servir de guide idéal pour mieux comprendre les fonctionnalités d’AEM.

{{ue-over-spa}}

Le SDK de l’éditeur de SPA AEM a été présenté dans une série de sessions AEM Gems. Ils ont été supprimés car l’éditeur de SPA est obsolète.
