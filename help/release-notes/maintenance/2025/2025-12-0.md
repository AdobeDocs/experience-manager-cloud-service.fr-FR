# Notes de mise à jour de maintenance {#maintenance-release-notes}

La section ci-dessous présente les notes techniques de mise à jour de la version de maintenance d’Experience Manager as a Cloud Service associées à l’activation de la fonctionnalité 2025.12.0.

## Version 23482 {#23482}

Vous trouverez ci-dessous un résumé des améliorations continues de la version de maintenance 23482, publiée le jeudi 3 décembre 2025. La version de maintenance précédente était la version 23385.

L’activation des fonctionnalités de la version 2025.12.0 fournit l’ensemble des fonctionnalités de cette version de maintenance. Voir [Feuille de route des versions d’Experience Manager](https://experienceleague.adobe.com/fr/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap) pour plus d’informations.

### Améliorations {#enhancements-23482}

* ASSETS-49770 : ajout de notifications de quarantaine pour les résultats de l’analyse des programmes malveillants.
* ASSETS-54079 : application d’un formulaire de métadonnées personnalisé au dossier de quarantaine.
* ASSETS-54083 : création d’un mécanisme de nettoyage des quarantaines planifiées.
* ASSETS-54278 : suppression de la propriété `dam:avScanTime` des ressources.
* ASSETS-57284 : limitation des chargements de fichiers au dossier Quarantaine (désactivation du glisser-déposer).
* ASSETS-57428 : masque le dossier de quarantaine dans l’interface utilisateur de l’affichage Assets.
* ASSETS-57626 : amélioration du comportement de reprise pour les tâches de ressources asynchrones.
* ASSETS-57879 : ajoutez une option de fusion pour les tâches asynchrones de déplacement/copie de ressources.
* ASSETS-58099 : ajoutez une configuration pour désactiver les balises intelligentes améliorées pour l’ensemble de l’environnement.
* ASSETS-58136 : implémentez les commentaires de pagination dans Search OpenAPI.
* ASSETS-59402 : ajouter des points d’entrée de tâche asynchrones pour l’API de suppression de dossier ; exporter des packages vers la région interne.
* ASSETS-59966 : renommez le groupe Administrateurs de programmes malveillants en Administrateurs de quarantaine.
* ASSETS-60166 : utilisez VideoViewer.js au lieu des URL basées sur iframe.
* GRANITE-61378 : Outil de débogage des autorisations - API ListPrincipals.
* GRANITE-63235 : requête d’identification des sites à l’aide de la propriété `cq:conf`, prend en charge la détection des anciennes pages/versions.
* SITES-30452 : API de contenu avec ASO - Suggestions de titre et de description, prise en charge de XWalk, opérations JSON Patch, liaison principale du service IMS.

### Problèmes résolus {#fixed-issues-23482}

* ASSETS-57430 : correction du chargement de la vue Assets sans prétraitement : exportation du package de `repoapi.preprocessing`, mise à jour de la RAPI vers la version la plus récente.
* ASSETS-58190 : réduire le total approximatif inutilement élevé dans l’interface utilisateur des collections.
* ASSETS-58866 : correction du titre/de la description/de l’ID de la ressource renvoyé dans les réponses OpenAPI.
* ASSETS-58868 : correction de la pagination lorsque les champs de tri sont manquants sur les ressources.
* ASSETS-58920 : correction du prétraitement de l’importation de ressources en bloc qui est ignoré.
* ASSETS-59168 : correction de l’heure de début/fin de l’analyse indiquant un fuseau horaire incorrect.
* ASSETS-59702 : correction de l’ordre des événements lorsque le statut de la ressource est défini sur Aucun statut.
* ASSETS-59830 : remise en file d’attente des tâches asynchrones arrêtées lors de la terminaison du pod.
* ASSETS-49757 : correctifs pour les événements d’analyse de détection de programmes malveillants.
* GRANITE-61019 : le correctif n’`gcMonitor` pas notifié lors de la première exécution après le redémarrage d’AEM.
* GRANITE-62717 : correction de la gestion des mots de passe `JSafe` avec des caractères non-ASCII.
* SITES-34331 : correction de la temporisation de chargement 503 de la superposition de déploiement pour les utilisateurs non-administrateurs (`cqLiveSyncCancelled index`).

### Problèmes connus {#known-issues-23482}

Aucun.
