Il comprend un module externe de blocs intégré qui peut afficher intuitivement une liste de tous les blocs aux auteurs, ce qui évite aux auteurs de mémoriser ou de rechercher chaque variante d’un bloc. Les développeurs peuvent également écrire leurs propres modules externes pour la bibliothèque sidekick.
