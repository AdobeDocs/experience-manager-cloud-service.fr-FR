# Utilisation de Adobe Experience Manager (AEM) Forms Designer {#install-designer}

Designer offre une interface graphique de conception de formulaires conviviale, qui simplifie la création de modèles de formulaires XPD. Vous pouvez concevoir un modèle de formulaire, définir sa logique et respecter des exigences légales strictes. Ces modèles servent de modèle de document d’enregistrement pour un formulaire adaptatif. Ils diffèrent des [modèles de formulaires adaptatifs](template-editor.md).
