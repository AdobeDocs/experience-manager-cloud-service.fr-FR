# Notes de mise à jour de maintenance {#maintenance-release-notes}

La section ci-dessous présente les notes techniques de mise à jour de la version de maintenance d’Experience Manager as a Cloud Service associées à l’activation de la fonctionnalité 2025.9.0.

## Version 22450 {#22450}

Vous trouverez ci-dessous un résumé des améliorations continues de la version de maintenance 22450, publiée le 16 septembre 2025. La version de maintenance précédente était la version 22171.

L’activation des fonctionnalités de la version 2025.9.0 fournit l’ensemble des fonctionnalités de cette version de maintenance. Consultez [Feuille de route des versions d’Experience Manager](https://experienceleague.adobe.com/fr/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap) pour plus d’informations.

### Nouvelles fonctionnalités {#new-features-22450}

* SITES-32595 : les workflows qui se terminent avec des fragments ignorés ou rejetés peuvent désormais être identifiés. Une nouvelle propriété est disponible dans la réponse API du workflow, qui répertorie les fragments exclus car non valides ou contenant des références non valides.
* SITES-33642 : un nouvel événement API est désormais produit et utilisé pour les fragments de contenu modifiés.
* SITES-33320 : il est désormais possible de rechercher un modèle de fragment de contenu à l’aide de son `technicalName` via l’API de recherche.

### Améliorations {#enhancements-22450}

* SITES-34023 : le champ `technicalName` a été ajouté aux réponses des points d’entrée des modèles de fragment de contenu pour une meilleure identification.
* SITES-32766 : les références des ressources de contenu dans les modèles de fragment de contenu prennent désormais en charge un plus large éventail de types de fichiers binaires.
* SITES-33974 : amélioration de la documentation OpenAPI pour la rendre plus précise et plus conviviale.
* SITES-9173 : mise en cache du `ContentPolicyStatus`.
* SITES-9290 : amélioration de la mise en cache du `TouchEditContext`.
* SITES-33355 : ouvre le nouvel éditeur de fragments de contenu sur « Afficher la payload » dans la console de workflow.
* SITES-33356 : ouvre le nouvel éditeur de fragments de contenu sur Créer un fragment de contenu → Ouvrir dans l’interface d’administration tactile.
* SITES-32952 : gestion incohérente des valeurs par défaut des champs de modèles de fragments de contenu lors de l’utilisation de l’API de diffusion.
* SITES-31539 : Edge Delivery avec l’éditeur universel : ajout de la prise en charge des balises meta de configuration de l’éditeur universel dans `head.html`.
* SITES-20672 : Edge Delivery avec l’éditeur universel : ajout de la prise en charge de feuilles de calcul de métadonnées en bloc supplémentaires dans l’instance de création.
* SITES-32963 : Edge Delivery avec l’éditeur universel : ajout de nouvelles métadonnées d’expérimentation pour la cible d’optimisation, l’affectation automatique et l’auto-apprentissage.
* SITES-30847 : publication des composants principaux 2.30.0.
* SITES-29617 : le point d’entrée referenceBy a été mis à jour pour utiliser la classe ReferenceSearch, ce qui améliore ses performances et sa fiabilité.
* SITES-19308 : amélioration des performances du processus de suppression de page en optimisant l’étape de validation des références.
* SITES-34293 : implémentation du chargement différé pour les ressources utilisées comme modèles afin d’améliorer les performances.
* SITES-33892 : ajout d’un bouton d’activation/désactivation de fonctionnalité permettant d’ignorer les vérifications des références pour les pseudo-pages, ce qui peut améliorer les performances.

### Problèmes résolus {#fixed-issues-22450}

* CQ-4360550 : correction d’une disparition inattendue de la copie linguistique après l’annulation d’un déplacement de page dans AEM Cloud Service.
* SITES-25232 : les liens Définir la date et Quitter la distorsion du temps ne disposent pas d’un indicateur de focus visible.
* SITES-25258 : le focus n’est pas géré avec la boîte de dialogue modale Supprimer l’annotation.
* SITES-25305 : la barre d’outils démographique ne reçoit pas le focus dans un ordre logique.
* SITES-25366 : le lecteur d’écran n’annonce pas le statut de chargement du modal de teaser.
* SITES-34276 : Edge Delivery avec l’éditeur universel : correction de la politique CORS créée automatiquement qui ne s’appliquait pas au niveau de la publication.
* SITES-34811 : Edge Delivery avec l’éditeur universel : correction du sélecteur hlx qui ne s’ajoutait pas aux liens vers les feuilles de calcul dans l’instance de création.
* SITES-31669 : chaînes non localisées « Cette page redirige vers » dans Outils > Sites > Lancements.
* SITES-30879 : chaînes non localisées dans Sites > Éditeur de page > Composant Recherche.
* SITES-30959 : chaînes non localisées dans Éditeur de page > Composant Image.
* SITES-21743 : chaîne « Veuillez sélectionner un document à afficher. » non localisée dans Éditeur de page > Visionneuse PDF.
* SITES-19785 : chaînes non localisées dans Composants principaux de site > Onglets.
* SITES-22059 : chaîne « Aperçu du fichier non disponible » non localisée dans Composants principaux de site > Visionneuse PDF.
* SITES-33360 : chaîne « Erreur au cours de l’opération. Le chemin d’accès fourni n’est pas un lancement » non localisée dans Lancements > Modifier.
* SITES-32975 : format de date non localisé dans Interface d’utilisation découplée > Lancements > Comparer le Lancement à la Source.
* SITES-32973 : chaînes codées en dur dans Interface d’utilisation découplée > Lancements > Rebase.
* SITES-13540 : chaînes non localisées dans Lancements > Promotion.
* SITES-13085 : chaînes d’erreurs non localisées dans Sites > Page de création de lancement.
* SITES-21499 : chaîne non localisée dans Sites > Lancements > Modifier.
* SITES-14961 : le champ de date est tronqué dans la boîte de dialogue Sites > Propriétés > Plan directeur > Déploiement.
* SITES-33764 : les filtres de lancement (chemin source/lancements créés par un workflow) ne fonctionnent pas.
* SITES-33884 : « Promouvoir la page active et les sous-pages » promeut accidentellement des pages non concernées.
* SITES-33611 : la vue d’ensemble de la Live Copy ne fonctionne pas pour les marchés à volume élevé.
* SITES-34331 : code 503 Timeout lors du chargement de l’écran de déploiement pour les utilisateurs et les utilisatrices sans privilèges d’administration.
* SITES-34403 : `NullPointerException` dans `GraphqlClientImpl deactivate()` lors de l’arrêt.
* SITES-33817 : résolution des problèmes de synchronisation entre le schéma d’interface d’utilisation et le modèle JCR pour garantir la cohérence.
* SITES-31141 : les références de contenu qui ne sont pas représentées par un chemin d’accès sont désormais correctement renvoyées dans la réponse de l’API.
* SITES-34080 : le processus de création de fragments de contenu est désormais plus robuste et n’échouera pas si aucun champ n’est fourni dans la requête.
* SITES-30773 : l’expression régulière permettant de rechercher des mots à l’aide de la fonction « Rechercher et remplacer » a été améliorée afin de correspondre correctement aux caractères UTF-8.
* SITES-33742 : résolution d’un bug qui empêchait le déplacement d’un fragment de contenu lors de l’utilisation de l’API de workflow.
