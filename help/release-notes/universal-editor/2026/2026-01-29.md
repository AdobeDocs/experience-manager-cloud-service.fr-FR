# Notes de mise à jour de l’éditeur universel version 2026.01.29 {#release-notes}

Voici les notes de mise à jour de la version du 29 janvier 2026 de l’éditeur universel.

>[!TIP]
>
>Si vous souhaitez tester les fonctionnalités de l’éditeur universel **à venir** avant leur publication, reportez-vous aux [Notes de mise à jour de l’aperçu de l’éditeur universel.](/help/release-notes/universal-editor/preview.md)

>[!TIP]
>
>Consultez [cette page](/help/release-notes/release-notes-cloud/release-notes-current.md) pour obtenir les notes de mise à jour actuelles d’Adobe Experience Manager as a Cloud Service.

## Nouveautés {#what-is-new}

* La méthode Supprimer le composant a été exposée à `editorActions`.
* La méthode du composant de copie a été exposée à `editorActions`.
* La méthode du composant Déplacer a été exposée à `editorActions`.
* La méthode Ajouter un composant a été exposée à `editorActions`.
* Les définitions étaient exposées à [extensibilité.](/help/implementing/universal-editor/extending.md)
* La version 2.0 de l’API de création de documents est publiée.

## Fonctionnalités d’adoption précoce {#early-adopter}

Si vous souhaitez tester les prochaines fonctionnalités répertoriées ci-dessous et partager vos commentaires, envoyez un e-mail à votre responsable du succès client Adobe à partir de l’adresse e-mail associée à votre Adobe ID.

* Une copie superficielle a été implémentée pour les fragments de contenu.

## Autres améliorations {#other-improvements}

* L’extensibilité `content` a été renommée `details` pour suivre le schéma d’API.
* Le comportement du menu contextuel a été amélioré.
* Correction d’un bug qui survenait lorsque l’utilisateur cliquait sur le badge d’un élément modifiable afin que le menu contextuel s’ouvre correctement.
